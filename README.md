# jinlon scan code fix
Fix the incorrect keyboard scan codes for HP Elite c1030 Chromebook / HP Chromebook x360 13c running linux.

## How to use
- Clone this repo or download and extract it
- cd into this repo
- Run 
```sh
chmod +x fix-keyboard.sh
./fix-keyboard.sh
```
